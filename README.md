## Google Books
A full-stack MERN app to search and save your favorite books.
<br>
<br>
Deployment: https://mern-googlebooks.herokuapp.com
<br>
Current bug in deployment: Search results not loading.
<br>
<br>
Technologies used:
<li>React
<li>Node
<li>Express
<li>MongoDB / Mongoose
<li>Google Books API
<li>Material-UI

Future enhancements:

<li>Login page
