# Google Books
A full-stack MERN app to search and save your favorite books.
<br/>
<br/>
Technologies used:
<li>React
<li>Node
<li>Express
<li>MongoDB / Mongoose
<li>Google Books API
<li>Material-UI
<br/>
<br/>
Future enhancements:
<li>Disable favorite button after you save it
<li>Login page
